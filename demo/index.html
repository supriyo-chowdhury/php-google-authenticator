2FA Integration Flow

How the five PHP files work together in your application.

1
setup_2fa.php Generates a secret, stores it in the session, renders the QR code and manual key.
2
enable_2fa_process.php Validates the OTP from the form, saves the secret to the DB, marks 2FA enabled.
3
verify_login_2fa.php Shows the OTP prompt during login (after password check passes).
4
verify_login_2fa_process.php Fetches the stored secret, verifies the OTP, rotates the session ID, redirects.
★
src/GoogleAuthenticator.php Core TOTP library — create secret, get code, verify code, generate QR URL.
